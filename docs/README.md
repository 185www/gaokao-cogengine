# 🧠 高考认知引擎 V3

基于学习科学的化学高考复习系统，采用双轨认知训练方法。

## 核心机制

- **客观验证 > 主观自评** - 关键词匹配与步骤校验，消除流畅性错觉
- **线索多样性** - 每张卡片含2-4种线索变体（直接/情境/逆向/对比）
- **过程反馈** - 常见错误详解 + 强制5秒停留，促进深度加工
- **渐进交错** - 新知识集中练习，熟练后解锁交错模式

## 三层知识深度

| 深度 | 名称 | 说明 | 数量 |
|------|------|------|------|
| 1 | 锚定 | 基础知识点记忆与理解 | 25张 |
| 2 | 链接 | 逻辑链推理与关联 | 12张 |
| 3 | 演练 | 综合解题实战（步骤校验） | 6张 |

## 六大模块

1. 氧化还原与离子反应
2. 化学反应原理
3. 无机元素化合物
4. 有机化学
5. 化学实验
6. 物质结构与周期律

## 技术特点

- 纯静态站点，零后端依赖
- 数据存储在浏览器 localStorage，离线可用
- 基于 SM-2 变体的间隔重复算法
- 渐进交错调度（新知识集中，旧知识交错）
- 暗色模式，移动端优先
